# 🌐 JavaScript Practice Projects

A collection of small, self-contained web projects built while learning JavaScript. Each one covers a different concept and is a standalone mini-app you can open right in your browser. No frameworks, no build tools, just plain HTML, CSS, and JavaScript.

---

## 📁 Projects

### 🧮 Calculator
A functional calculator that handles basic arithmetic operations. Good practice for DOM manipulation and handling user input events.

### 🎲 Dice Roll
Click a button and roll a dice. A simple project that covers random number generation and dynamically updating the UI.

### 🕐 Digital Clock
A live clock that updates every second using JavaScript's `Date` object and `setInterval`. Shows hours, minutes, and seconds in real time.

### ✂️ Rock Paper Scissors
Play against the computer. The game picks randomly and tells you who won. A nice exercise in conditionals and game logic.

### ⏱️ Stopwatch
A working stopwatch with start, stop, and reset controls. Covers timing functions and managing application state.

### 🌡️ Temperature Converter
Convert temperatures between Celsius, Fahrenheit, and Kelvin. Clean form handling and formula logic in action.

### ☁️ Weather App
Fetches live weather data using a weather API and displays it for a given city. This one introduces working with APIs, `fetch`, and handling async data in JavaScript.

## 🚀 How to Run Any Project

No installation needed. Just:

1. Clone the repo:
   ```bash
   git clone https://github.com/radhika-khatri/JavascriptPracticeProjects.git
   ```
2. Open the folder of the project you want to try.
3. Double-click `index.html` to open it in your browser.

That's it.

> For the Weather App, you'll need to plug in your own API key from [OpenWeatherMap](https://openweathermap.org/api) or whichever weather API is used in the project.
